# DPE Certificate Parser and Verification

This directory contains Python scripts to parse and verify X.509 certificates generated by DPE. It uses the `cryptography` and `asn1crypto` libraries for parsing and verification.

## Prerequisites

*   [Rust and Cargo](https://www.rust-lang.org/tools/install)
*   [uv](https://github.com/astral-sh/uv) (a fast Python package installer and resolver)

## Setup

### Using script

You can run the `![verification script](./run_tests.sh_) to run the verifier tests.

### Manual Steps

1.  **Set up the Python Environment:**
    Navigate to this directory (`verification/cert_parser`) and create a virtual environment and install the required packages using `uv`.

    ```bash
    uv venv
    uv pip install -e .
    ```
    This will create a `.venv` directory and install the packages listed in `pyproject.toml`.

## Running the Tests

Once the setup is complete, you can run the integration tests. The tests will automatically run the `sample_dpe_cert` tool, generate certificates, and verify them.

From within the `verification/cert_parser` directory, run:

```bash
.venv/bin/pytest
```
